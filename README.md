## 1. Bidirectional Centripetal Insertion Sort (BCIS)
A novel sorting algorithm that optimizes the classic insertion sort by processing elements from both extremities (Left and Right) simultaneously, converging at the center.

### Performance:
- **Best Case:** O(n)
- **Average/Worst Case:** O(n^2)
- **Space Complexity:** O(1)

---

## 2. Bidirectional Linear Search (BLS)
An optimized search mechanism that scans the data structure from both ends concurrently. This reduces the number of iterations by 50% in the worst-case scenario compared to traditional linear search.

### Performance:
- **Best/Worst Case:** O(n/2) which simplifies to O(n)
- **Space Complexity:** O(1)
